# self-release-notes

## 概要
- ["じぶんリリースノート" と称した月報を続けて3年が経った](https://blog.a-know.me/entry/2019/02/02/214612) のルールを参考
- 毎月月初に新しい自分がリリースされたとして更新。月報がわり。
- バージョン番号は 0.年齢.月齢 とする。メジャーバージョンアップは三途の川を渡ったときとする。
